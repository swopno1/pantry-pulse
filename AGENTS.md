# PantryPulse Development Guidelines

- **Stack**: React (Vite), Tailwind CSS v4, Lucide React.
- **Design System**:
  - Sage Green: #4F7942
  - Warm Amber: #D97706
  - Charcoal: #1F2937
  - Off-White: #F9FAFB
  - Fonts: Inter (Sans), Playfair Display (Serif)
- **Rules**:
  - Keep the app zero-cost/frontend-only.
  - Store API keys in localStorage.
  - Maintain a clean, minimalist aesthetic.
